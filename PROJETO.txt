💻 TECNOLOGIAS UTILIZADAS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔧 Backend
   • Node.js + Express.js      → Servidor HTTP
   • Sequelize                 → ORM para banco de dados
   • SQLite3                   → Banco de dados local
   • JWT                       → Autenticação
   • bcryptjs                  → Hash de senhas
   • CORS                      → Segurança cross-origin

🎨 Frontend
   • React 18                  → UI Library
   • Vite                      → Build tool
   • Axios                     → HTTP Client
   • React Router              → Roteamento (preparado)
   • CSS Puro                  → Estilos (azul marinho #003366)

💾 Database
   • SQLite3                   → Banco de dados sem servidor
   • 3 Tabelas                 → users, categories, transactions

├── nome
├── email (UNIQUE)

categories
├── id (PK)
├── userId (FK)
├── nome
└── tipo (entrada/saída)

transactions
├── id (PK)
├── userId (FK)
├── categoryId (FK)
├── valor
├── descrição
└── data

1. LoginForm.jsx
   → Tela de login e registro
   → Toggle entre login/registro
   → Validação de email e senha

2. Dashboard.jsx
   → Tela principal
   → Cards de resumo (entrada, saída, saldo)
   → Seletor de data
   → Botão gerenciar categorias

3. TransactionForm.jsx
   → Formulário para registrar transações
   → Dropdown de categorias dinâmicas
   → Campo de valor, data e descrição
   → Submit e validação

4. TransactionTable.jsx
   → Tabela com todas as transações do dia
   → Exibe hora, categoria, tipo, valor, descrição
   → Botão delete para cada transação
   → Mensagem quando não há transações

5. CategoryManager.jsx
   → Modal para gerenciar categorias
   → Criar novas categorias (entrada/saída)
   → Listar categorias por tipo
   → Deletar categorias existentes
